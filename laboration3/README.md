
# Laboration 3: Vue App

En single-page-application skapad i ramverket Vue som använder ett separat API skapat med Hapi som hanterar data
från en MySQL-databas i form av böcker.
<br>
Webbplatsen tillåter användaren att läsa, lägga till och radera inlägg från databasen via webbplatsen.
<br>
Informationen lagras i databasen på följande sätt:
- name (titel)
- year (utgivningsår)
- read (läst/ej läst, i form av ett true/false-värde)
